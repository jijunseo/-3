🚀 초기 설정 순서
  1. 봇을 서버에 초대 (관리자 권한)
  2. /backup save — 첫 번째 백업 생성
  3. /autobackup enable — 자동 백업 켜기
  4. /autorecovery enable — 자동 복구 켜기
  5. /auth setup — 인증 채널 설정
  6. /auth panel — 인증 버튼 전송
  7. /shop setup — 자판기 채널 설정
  8. /shop panel — 자판기 패널 전송
  9. 웹 패널에서 상품 등록 후 완료! ✅
🚨 서버 복구 방법
  1. 새 디스코드 서버 생성
  2. 봇을 새 서버에 초대 (관리자 권한)
  3. /restore latest 입력
  4. 채널·역할이 자동 복구됩니다
  5. 웹 패널 → 멤버 관리 → "전원 재참가" 클릭
  6. 인증한 멤버들이 자동으로 재참가합니다 ✅
⚠️ 재참가는 인증 버튼을 눌러 OAuth2 토큰이 저장된 멤버만 가능합니다.
